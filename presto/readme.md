- 统计注册 IP 数
- 统计每日访问 IP 数
- 统计页面实时 UV 数

- 统计用户每天搜索不同词条的个数
- 流量监控（多少不同IP访问过一个服务器）
- 数据库查询优化（例如我们是否需要排序和合并，或者是否需要构建哈希表）
